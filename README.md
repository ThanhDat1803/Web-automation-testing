# 📌 Web Automation Testing Project

## 📖 Description
This project is an automated web testing framework designed to validate core functionalities and UI elements of a web application. It supports reusable test execution, reporting, and log generation for analysis.

---

## 🚀 Features
- Functional testing:
  - Login
  - CRUD: TestCase, TestPlan, SRS
  - Dropdown, Table Headers
  - File download
- UI testing:
  - Font, color, layout, images
- Data-driven testing using JSON / Excel
- Automated test execution & reporting

---

## 🛠️ Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven / Gradle
